# untappi-appi
Find top beers from nearby bars with the help of Untappd APIs. Originally hosted on AKS (Azure Kubernetes Service), but now on Civo managed k3s Kubernetes cluster. 

### Frontend
* React
* tailwindcss

### Backend
* .NET Core 3.0
* Entity Framework Core
